# 🚨 RoadSoS

RoadSoS is a modern emergency response web application that helps users quickly access emergency support during critical situations.

The app includes:
- 🚨 SOS emergency alerts
- 📍 Live location tracking
- 🎤 Voice activated SOS
- 🚑 Emergency services
- 📞 Quick emergency calling
- ⚙️ Smart settings system
- 🔋 Battery & live time status
- 🌙 Dark mode support

---

# 🌟 Features

## 🚨 SOS Emergency System
- One tap SOS activation
- Emergency popup alert
- Flash warning animation
- Emergency siren sound
- Device vibration support

---

## 📍 Live Location
- Detects current location
- Shows latitude & longitude
- Embedded Google Maps
- Nearby hospital support

---

## 🎤 Voice SOS
Activate SOS using voice commands:
- "Help"
- "SOS"
- "Emergency"

---

## 🚑 Emergency Services
### Included Modules:
- Vehicle Assistance
- Ambulance Services
- Police Assistance
- First Aid Guide

---

## 📞 Emergency Call System
Quick emergency calling:
- 👮 Police → 112
- 🚑 Ambulance → 108
- 🔥 Fire Brigade → 101

---

## ⚙️ Settings
- 🌙 Dark mode toggle
- 📱 Live battery status
- ⏰ Live clock
- 📞 Emergency contact editing

---

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/RoadSoS.git
```

---

## 2️⃣ Open Project

```bash
cd RoadSoS
```

---

## 3️⃣ Install Flask

```bash
pip install flask
```

---

## 4️⃣ Run Application

```bash
python app.py
```

---

## 5️⃣ Open Browser

```text
http://127.0.0.1:5000
```
